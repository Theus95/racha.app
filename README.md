# racha.app

> Organiza. Convoca. Joga.

## Como rodar localmente

```bash
npm install
npm run dev
```

Abre em `http://localhost:5173`

## Como fazer deploy no Vercel

1. Sobe essa pasta no GitHub
2. Acessa vercel.com → "Add New Project"
3. Importa o repositório
4. Clica em Deploy

Pronto. URL gerada automaticamente.

## Domínio personalizado

No Vercel → Settings → Domains → Add "racha.app"
Configure o DNS no seu registrador (Registro.br, GoDaddy, etc.)
